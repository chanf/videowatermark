# VideoClean - 专业视频去水印工具

一个基于 Web 的专业视频去水印工具，支持多种填充方式和实时预览。

## 功能特性

- 🎬 **视频上传与预览**：支持 MP4、MOV、AVI、MKV 等多种视频格式
- 🖼️ **帧截取与选区**：精确截取视频帧并框选水印区域
- 🎨 **多种填充方式**：
  - 纯色填充：使用自定义颜色覆盖水印
  - 图片填充：上传图片覆盖水印区域
  - 智能填充：AI 推断周围内容（开发中）
- 👁️ **实时预览**：在处理前预览效果
- ⚡ **浏览器内处理**：使用 FFmpeg.wasm 在浏览器中直接处理视频，无需上传服务器

## 技术栈

- **框架**：React 18 + TypeScript
- **构建工具**：Vite
- **样式**：Tailwind CSS
- **视频处理**：FFmpeg.wasm
- **Canvas**：用于帧处理和填充效果

## 开始使用

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:5173` 启动。

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 使用说明

1. **上传视频**：拖拽或点击上传视频文件
2. **选择参考帧**：播放视频到合适位置，点击"截取当前帧"
3. **框选水印**：使用鼠标在截取的帧上框选水印区域
4. **选择填充方式**：
   - 纯色填充：选择颜色
   - 图片填充：上传覆盖图片
   - 智能填充：自动推断内容
5. **预览效果**：点击"预览效果"查看处理前后对比
6. **开始处理**：点击"开始处理"生成新视频
7. **下载**：处理完成后下载去水印后的视频

## 注意事项

1. **浏览器兼容性**：需要支持 SharedArrayBuffer 的现代浏览器（Chrome、Firefox、Edge 等）
2. **文件大小**：由于在浏览器中处理，建议使用较小的视频文件（< 500MB）
3. **处理时间**：视频长度和分辨率会影响处理时间
4. **内存使用**：视频处理会占用较多内存，建议关闭其他标签页
